# Seti UI Red

[Fork of the beautiful seti-ui theme by Jesse Weed]

![Screenshot](https://github.com/kryptonianson/seti-ui-red/raw/master/screenshot.png)


### Installation

```
cd ~/.atom/packages
git clone https://github.com/kryptonianson/seti-ui-red --depth=1
```
